### Technologies

* Bootstrap 4.5.0
* Webpack 5.0.0-beta.22
* JavaScript 
* jQuery
* Font Awesome Icons 5.14.0

### Installation 

This app requires [Node.js](https://node.js.org/) v14.4.0 to run

Install the dependecies

```sh
$ npm install
```

Start the dev server

```sh
$ cd photography-portfolio-site
$ npm run serve
```

For production build

```sh
npm run build
```
